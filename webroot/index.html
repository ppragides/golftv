You've successfully set up GolfTV

If you've gotten this far, that means that you were able to set up your environment such that you're able to serve out the TVML required for our tvOS application.

Now what?

All you'll need to do is to create a new tvOS project, and set your application's tvBaseURL to your local minikube instance.

If you've followed all of the instructions in the README.md file, it should look something like this:

static let tvBaseURL = "http://local.golf.tv:30080/"
